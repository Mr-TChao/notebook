
编辑模式切换到命令模式
```
Esc
```

命令模式快捷键
```
F: 查找并替换
P: 打开命令配置
Y: 把代码块变成代码
M: 把代码块变成标签
R: 清除代码块格式
K 或 ⬆ : 选择上面的代码块
J 或 ⬇ : 选择下面的代码块
A: 在上面插入代码块
B: 在下面插入代码块 
X: 剪切选择的代码块
C: 复制选择的代码块
V: 粘贴到下面
Z: 撤销删除
S 或 Ctrl + S: 保存并检查
L: 切换行号
O: 选择单元格的输出
H: 显示快捷键
Q 或 Esc: 关闭页面
Enter： 进入编辑模式
Ctrl + Shift + F/S ： 打开命令配置
Shift + Enter： 运行代码块并选择下方代码块
Ctrl + Enter： 运行当前代码块
Alt + Enter： 运行当前代码块并在下方插入新的代码块
Shift + K/↑： 扩展上面选择的代码块
Shift + J/↓： 扩展下面选择的代码块
Shift + V：粘贴到上面
Shift + M：合并选中单元格，如果只有一个单元格被选中则将其与下一个单元格合并
Shift + O：切换选中单元格的输出滚动
Shift + L: 在所有单元格中切换行号，并保持设置
双击D: 删除选中单元格
双击I：中断服务
双击O: 重启服务（带窗口）

```

编辑模式快捷键
```
Esc：进入命令模式
Tab: 代码完成或缩进
Insert: 切换重写标志
Shift + Tab: 工具提示
Shift + Enter: 运行当前代码框并选择下方代码块
Ctrl + ] : 缩进
Ctrl + [ : 取消缩进
Ctrl + A : 全选
Ctrl + Z : 撤销
Ctrl + / : 评论
Ctrl + D : 删除整行
Ctrl + U : 撤销选择
Ctrl + Home/ ↑ : 跳到单元格起始处
Ctrl + End/↓ :跳到单元格最后
Ctrl + ← : 跳到单元格左边
Ctrl + → : 跳到单元格右边
Ctrl + 删除键: 删除前面的单词
Ctrl + delete: 删除后面的单词
Ctrl + Y : 重做
Ctrl + M : 进入命令行模式
Ctrl + Enter: 运行选中代码块
Ctrl + S :保存并检查
Alt + U : 重新选择
Ctrl + Shift + F/P : 打开命令配置
Alt + Enter : 运行当前代码块并在下方插入新的代码块
Ctrl + Shift + minus : 在光标处分割代码块

```
