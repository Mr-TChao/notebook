# 影刀RPA -- 逻辑思维
## 逻辑思维上 -- 再探循环
不同循环方式的使用场景

| 指令          | 场景                 |
| ----------- | ------------------ |
| ForEach列表循环 | 用于循环遍历列表中的每一项数据    |
| For次数循环     | 用于按照固定次数循环操作       |
| 循环Excel内容   | 用于循环Excel表格的内容     |
| 循环数据表格内容    | 用于逐行遍历影刀内置数据表格中的内容 |
| 循环相似元素(web) | 用于循环操作网页上的相似元素     |
| 循环相似元素(win) | 用于循环操作软件中的相似元素     |
| 无限循环        | 用于执行不固定次数的循环操作     |
